# 🧠 RAG Assistant

A powerful AI-powered document assistant built with LangChain, FAISS, Groq, and Streamlit.

## 🌐 Live Demo
[Click here to try the app](https://rag-app-5yqebotjy2vmjrqyjom3cq.streamlit.app)

## ✨ Features
- 📄 Upload multiple PDF and TXT files
- 🌐 Chat with any website by pasting a URL
- 🔍 Semantic search across all documents
- 🤖 AI answers grounded in your content
- 💬 ChatGPT-style typing animation
- 📌 Source viewer with file and page info
- 🎨 Beautiful dark UI

## 🛠️ Tech Stack
- **LangChain** — RAG pipeline
- **FAISS** — Vector database
- **HuggingFace** — Free embeddings
- **Groq (Llama 3.1)** — Free LLM
- **Streamlit** — UI framework

## 🚀 How to Run Locally
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your `GROQ_API_KEY` in a `.env` file
4. Run: `streamlit run app.py`
